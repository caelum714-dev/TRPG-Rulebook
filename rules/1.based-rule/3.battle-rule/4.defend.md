# 防御手段

::: details 防御等级

## 防御等级

所有攻击都有对应的伤害类型，而你的残壳，被动与护甲会为你提供相对的不同类型的防御等级。

当你特定的防御等级大于目标造成伤害类型的冲击力时，你受到的伤害减半，当超过3级时，你不受到伤害。低于三级时，你还会额外扣除等同于伤害的生命值，低于6级时，你会直接被重创。敌人同理，且没有重创状态的杂兵会被直接秒杀。防御等级为0时，对应的冲击不会造成护甲值伤害，而是直接造成生命值伤害。

:::

::: details 掩护

## 掩护

可以在自己的回合消耗一个【动作】并令角色寻找最近的掩体（20尺之内）。宣言一个周遭环境中可能可以当作掩体的事物，若主持人觉得合理的情况下，并在下一个回合开始前使自己的角色处于掩护状态。一般来说，只要宣言的不是太过特殊的物件，一名玩家宣告后，其余玩家都可以在这个场景中宣告同样的掩体。以下是掩体的质量与效果：

·**这片区域不可能存在掩体**

无效果。

·**脆弱的或不完全的掩体**

例如踢倒桌子或者举起小型盾牌，或者站在你面前的不持有大型掩体的人类：使你的角色处于【难以命中】状态（1级掩护，必须打出扑克才能将你选为攻击目标）。

·**覆盖的或坚固的掩体**

例如大型路障或展台基座：使你的角色处于【不可命中】状态（2级掩护，无法选为攻击目标）。

**※**：原先就存在的墙体并不属于掩体的范畴。掩体是场景中可能存在的物件，不存在方向的概念（你的角色没有蠢到呆在掩体一侧不动且不旋转这些掩体）。

值得注意的是，场景中的某些墙体若低于你的高度，可能只能算作不完全掩体。同时，除去盾牌一类完全易于掌控的掩体，绝大部分掩体不能为你提供近程攻击的掩护。

:::
