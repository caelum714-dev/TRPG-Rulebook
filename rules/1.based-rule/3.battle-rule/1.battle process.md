# 战斗流程

::: details 流程描述

## 流程描述

场上所有单位全部进行完毕自己的回合并经历清算阶段的情况下被称为完成了一轮。

在战斗开始时，敌方的【护甲值】和本轮使用的【反制】会向玩家进行展示。随后，玩家自行进行行动，并依次汇报关键行动与造成的伤害总和。或者也可以按照玩家自定的顺序依次行动。

在战斗开始时，敌人仅一次先行进行准备行动。在此阶段，敌人会设下诈唬和反制。

在玩家完全行动后，敌人发起行动。

在敌人行动结束后，进行清算阶段，集中结算延迟效果。例如手雷或部分蓄力技能等。

:::

::: details PVP

## PVP

在PVP发生时，根据现场情况决定先手的对象。难以判断的情况下，丢硬币判断先后手。后手的团队如敌方单位一样，在没有被伏击的情况下可以在准备阶段使用诈唬。

:::

::: details 时点

## 时点

1.战斗开始（仅一次）

应用战斗开始时效果。

2.准备阶段（仅一次）

敌人或后发队伍设下诈唬和反制。

3.玩家行动

4.敌人行动

5.清算阶段

集中结算延迟效果。

6.战斗结束（仅敌人或某一方全部撤离/阵亡时触发一次）

应用战斗结束时效果。

:::

::: details 伏击/高速战斗

## 伏击/高速战斗

在成功布下埋伏或敌人过于弱小时，应用【高速战斗】规则。

玩家可以按照自定的顺序依次抽牌，并视抽出牌的点数为武器的点数，武器应用的属性价值作为追加伤害（见后文进攻手段）对目标发起一次攻击。

在所有玩家进行过一轮攻击后，可以进行下一轮攻击。

若最终伤害点数小于目标挑战等级，或抽牌达到手牌上限，则这名玩家之后将不能在【高速战斗】中发起攻击。在所有玩家均无法攻击后，进入常规战斗阶段。

在高速战斗触发的情况下，准备阶段会被移除。

若在战斗当中发起偷袭的情况下，仅发起偷袭的那个对象应用此规则进行【高速战斗】，整个高速战斗环节视为消耗一个动作。

:::

::: details 决斗

## 决斗

在1V1决斗或希望简化，快速结束战斗时，可以应用【决斗】规则。在这个特殊规则中，玩家无视地图和距离规则，而是每轮尽可能将手牌抽至上限。在这种情况下，玩家的动作和快速动作不再能用于施展奇术，而奇术则不再消耗任何动作资源。在此情况下可以尽可能快速结束死斗。

:::
