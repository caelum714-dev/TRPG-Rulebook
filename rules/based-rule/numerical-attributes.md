::: details 生命值

## 生命值

角色的生命值。生命值清空后你的角色就该嗝屁了，简单好懂。但说是直接嗝屁，你的角色死前肯定还是得挣扎一下。因此实际上在生命归0时你的角色会进入【重创】状态

你的初始生命值为20+体力

:::

::: details 移动力

## 移动力

你的角色每回合可以移动的距离为40m。（10m为一格）

若你的移动方式不常规（爬行）或你正经过某些难以穿越的区域，则需要消耗两倍的行动力移动相同的距离。（消耗20m的行动力移动10m的距离）

:::

::: details 真理值（VERACITY）

## 真理值（VERACITY）

你的角色对现实的掌控力。对于非施法者来说，真理度的降低代表了不稳定与冲动的行为。而对于施法者来说，真理度的降低某种意义上代表了在他们眼中，现实正在变得柔软且可以塑造。这允许了他们释放更为强大的法术，也令他们离失控更近一步。

你的初始真理值20+理性

在你的角色真理值为满时：

你的角色认知完备，行为正常。

在你的角色真理值剩余一半时：

你的角色认知开始出现偏差，但不会有实质惩罚。

在你的角色真理值剩余五分之一时：

你的角色在进行检定或战斗开始时，需要进行理性检定。失败的情况下你的角色受到【癫狂】状态影响。

在你的角色真理值归零时：

你的角色进入【癫狂】状态，直到理智恢复到至少一半。且再扣除真理值时，需要进行理性检定，否则会获得长期癫狂的影响或立即失控。

值得注意的是，若你的角色灵性低于3，则不拥有真理值这一属性，也不会受到这个属性带来的任何影响。

:::
