# 维度

你可能会希望你的角色拥有多种多样的复杂能力，但其中许多基础能力的有机结合才能达成，另一部分则可能多项基础能力都能带来类似的效果。例如卓越的发明能力可能需要学识，智力与灵性的结合；而无论是体力，体术还是智力都能为你的角色带来某种锻炼方式让你自身更加强壮。因此，在引入这些多样的能力前，我们需要进一步归类划分先前的本格属性，将其划入不同的【维度】（行与列），如下表所示:

在此基础上，满足某些规则便可以选择携带某项被动。例如【维度（口）吸引力】要求：基石（口）点数投入和≥12。随后，在你满足投入要求的情况下，例如体力5，理性5，魅力2，你便可以获得其带来的益处：在进行试图引起对应维度属性的角色好感的检定中，享受双倍属性加成。这时你可能会皱起眉头：不对啊，这魅力只有2，“开口就不讨喜，人厌狗嫌”，怎么，还享受上带来好感的加成了？但此时，我们不能忽略卓越的体魄和理性带来的自然魅力。生活中我们也常见有许多言行举止非常惹人生厌，行为利己冷酷但身体健壮外貌帅气的衣冠禽兽之人，许多人还真就被这种所谓的“坏人的魅力”所吸引了。这便是“多维自我”系统存在的意义，让你可以不用纠结于用宽泛的基础属性去定义你的角色，而是可以让你的角色更加多维与立体。
