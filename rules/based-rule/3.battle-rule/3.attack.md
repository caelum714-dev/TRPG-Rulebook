# 进攻手段

::: details 攻击

## 攻击

消耗一个【行动】在进行普通的近程或远程攻击时，依据武器伤害记录的骰子骰点决定基础点数。随后加入武器的关键属性加值，得到最终点数。

每次你的回合开始时或你的武器用骰子命中时，你可以抽取一张牌加入手牌。

你也可以选择不投骰子而是打出任意的手牌或者底牌来计算基础点数，但只有满足某种牌型的牌才会计入点数。例如，你同时打出了一张8和一张A，那么会被视为高牌。只有A会结算，即点数仅为1。

但若打出两张6则会视为对子，即6+6=12点数。若打出小鬼，则将牌型直接视为小鬼。若打出大鬼，则视为大鬼牌型，或其它任何牌型。

**战斗对抗**：骰子与骰子，或骰子与牌对抗的情况下，将比较最终点数的大小。只有符合牌型的牌才会被计入点数。在特定情况，如特定战术下，会直接比较牌型大小而无视点数。

**群体对抗**：有些时候，会出现某个目标的攻击覆盖了多个目标的情况，在此情况下有可能会发生群体对抗。那么如何判断这种情况下对抗的输赢呢？判断的主要依据便是攻击有没有脱手。若对手造成破坏的武器与自己的身体有相连的地方，例如流星锤或剑，则依次对抗。只要有一人对抗成功剩余单位便不需要承担效果。在武器脱手，例如打出炮弹的情况下，则范围内所有人都要参与对抗，所有失败者均要承受后果。

**击溃**：对抗失败的一方或护甲值（见后文）在某次攻击中被清零的一方将被击溃。击溃没有实质负面效果，但击溃对手时可以对被击溃的对手施加一些负面效果。

**冲击力**：你的武器会有一个基础的冲击力，而你的被动等会增加这个冲击力。这个数据直观反应了武器的破甲能力和破坏力，用以对抗对手的防御等级（见后文）。

**追加伤害**：这个名词描述的数值不会计入点数，而是命中后作为加值加入造成的伤害中。同样，也会被防御等级影响（见后文）。例如，你的武器描述了追加5点伤害，攻击时骰点骰出了11。那么你会造成16的伤害，但若发生对抗且对方点数大于11那么你会败于对抗。同样的，若你的冲击力小于对手的防御等级，你只会造成（11+5）÷2=8点伤害。

::: details 奇术

## 奇术

消耗指定的行动类型，打出特定的牌型后，可以触发某些奇书的效应。详见奇术书。

:::

::: details 诈唬

## 诈唬

你在凑齐了自己想要的牌后，也可以不立刻将它们作为奇术打出，而是消耗一个动作将其盖下。他人可以看见你盖下卡牌的数量。在此基础上，其它人将你选作攻击目标时，你用盖下的牌触发一个牌型并与对手对抗（这意味着你可以盖下远超战术所需的牌）。成功的情况下立刻回到你的回合，即使它已经结束。在这个临时回合当中，你可以继续执行你剩余的行动，以及在你想要的时机释放你刚刚用于对抗的战术。在你结束你的回合后，对方将继续他的回合。你的盖牌没有被触发的情况下，你收回这些牌。触发的情况下，所有这些牌都会被丢弃。

:::

::: details 衍生物操控

## 衍生物操控

你所有的衍生物公用一个牌堆，这个牌堆被称为【操控力】。在你满足要求出牌创造新的衍生物时，这些牌会置于你的手牌边。若你制造了超过1个衍生物时，这些牌会叠放在一起。

:::
